Secondary memory - Long Term Memory
− Secondary memory in computers refers to non-fargile storage devices like hard disk drives and solid-state drives, preserving data even when the computer is powered off.

What is Secondary Memory and what does it do?

Secondary memory in computers refers to storage devices that are used to store data and programs permanently. Unlike primary memory (RAM), which is volatile and loses its data when the computer is turned off. Secondary memory retains data even when the power is turned off. Secondary memory is non-volatile and is used to store large amounts of data, software applications, and operating systems.

How Secondary Memory works

When you save a file or install a program, the data is transferred from the computers primary memory (RAM) to the secondary memory, such as a hard disk drive or solid-state drive. These storage devices use magnetic or electronic technology to encode and store data in binary format. When you need to access the stored data, the computerss processor retrieves the information from the secondary memory device and loads it back into the primary memory for processing. This process allows users to store large amounts of data and ensures that the data remains intact even after the computer is powered off and restarted.

File Storage

Secondary memory is essential for tasks such as data backup, file storage or running large applications that require significant storage space. When you save a file on your computer it is typically stored in secondary memory, ensuring that the data is preserved even if you shut down your computer.

Measurments

Bytes: The smallest unit of data storage. A byte is composed of 8 bits.

Kilobytes (KB): Approximately a thousand bytes.

Megabytes (MB): Approximately a million bytes or a thousand kilobytes.

Gigabytes (GB): Approximately a billion bytes or a thousand megabytes.

Terabytes (TB): Approximately a trillion bytes or a thousand gigabytes.

Petabytes (PB): Approximately a quadrillion bytes or a thousand terabytes.

Exabytes (EB): Approximately a quintillion bytes or a thousand petabytes.

Solid Date Drives (SSD)

An SSD or Solid State Drives is a storage device that uses flash memory without moving parts, offering fast data access and improved durability compared to traditional spinning disks.

Hard Disk Drive (HHD)

An HDD or Hard Disk Drive, is a storage device that uses magnetic storage to store and retrieve digital information through rapidly rotating disks.
